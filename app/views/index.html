Create prototype code for the GOVUK prototype kit

This is a tool to create Nunjucks code to be used in the GOVUK Prototype Kit. You can use the prompt to request various page templates and all the components from the GOVUK Design System.

It will create a preview of what the code will do, the Nunjucks code to paste into your code editor and a copy code button to copy the code to your clipboard.

This tool is not powered by Ai. Cloning the GitHub repository will allow you to run the kit the same as any other GOVUK Prototype Kit and does not require an internet connection to run
